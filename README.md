# Contao 4 firefighter bundle

With the Firefighter Bundle, the members of a fire brigade can be conveniently recorded and processed with their specific data. The data that can be recorded includes name, rank, functions, trainings and honors / awards. In addition, a content element for operational resources is provided.
Visit the [project website][1] for more information.
